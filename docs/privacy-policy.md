### Privacy Policy

Last updated: January 2026

### Overview

This Chrome extension is designed to enable remote control of media playback within the user’s browser. We respect user privacy and are committed to transparency regarding data usage.

This extension does not collect, sell, or share personal data.

### Data Collection

The extension does not collect any personally identifiable information, including but not limited to:
- Name, email address, phone number
- Browsing history or search history
- Website content (text, images, forms, or inputs)
- Credentials, passwords, or payment information

### Limited Technical Data Usage
To function correctly, the extension temporarily processes non-personal technical data, including:
- Active media tab identifiers
- Media playback state (e.g., playing, paused, muted)
- Session identifiers and trusted device tokens

<b>This data: </b>
- Is used only within an authenticated session
- Is not shared with third parties
- Is not used for analytics, tracking, or advertising

### Data Storage

The extension uses Chrome’s storage API to store:

- Connection state
- Session or pairing identifiers
- User preferences related to the extension

<b> All stored data: </b>
- Remains on the user’s device
- Can be cleared by removing the extension or clearing browser data
- Is never transmitted for profiling or analytics

### Network Communication

The extension may communicate with a user-controlled or trusted server using secure WebSocket connections for media control synchronization.

<b>Important clarifications:</b>
- No remote JavaScript or executable code is downloaded or executed
- Network communication is limited to structured control messages
- No personal data is transmitted

### Permissions Usage

Permissions requested by the extension are used strictly to:
- Detect media-capable tabs
- Control media playback based on user action
- Maintain session continuity

<b>Permissions are not used to monitor general browsing activity.</b>

### Third Parties

This extension:
- Does not use third-party analytics
- Does not integrate ads or trackers
- Does not sell or share data with any third party

### User Control

Users can:
- Disconnect remote sessions at any time
- Remove the extension to delete all stored data
- Control when the extension is active

<b>The extension remains inactive unless explicitly connected by the user.</b>

### Changes to This Policy

If this privacy policy changes, the updated version will be published with a new extension release.

### Contact

If you have questions or concerns about privacy, you may contact the developer through the Chrome Web Store support channel associated with this extension.
